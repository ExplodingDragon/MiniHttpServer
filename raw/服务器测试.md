# 服务器测试

> Http 服务器下的测试工具，用于测试此服务器是否工作正常

## 服务器信息

> 关于服务器的基本配置信息

**绑定端口**：``` <PORT/>```

**系统属性**：

```bash
<PROPERTY/>
```

**请求的IP**：```<IP/> ```

**绑定的解释器**：

```bash
<BINDER/>
```

## GET 测试

> 测试 GET 请求是否正常，此选项只会在 GET 下显示有效内容

| KEY  |  VALUE  |
| :--: | :-----: |
|      |         |
<GET_ARRAY/>
